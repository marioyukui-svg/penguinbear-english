词性转换练习

一、什么是词性转换？

同一个单词可以变成不同的形式，用来表达不同的意思。

  • 形容词 → 名词：happy → happiness（开心 → 开心的事）
  • 形容词 → 副词：quick → quickly（快的 → 快速地）
  • 动词 → 名词：teach → teacher（教 → 老师）

二、常见转换规则

转换类型规则例子
形容词 → 副词大部分加 -lyslow → slowly
形容词 → 名词加 -nesskind → kindness
动词 → 名词（人）加 -er / -orteach → teacher

三、练习题

第1题：形容词变副词

把下面的形容词变成副词（加 -ly）。

  1. quick → ________
  2. slow → ________
  3. happy → ________（注意：y 变 i 再加 -ly）
  4. loud → ________
  5. careful → ________

第2题：形容词变名词

把下面的形容词变成名词（加 -ness）。

  1. kind → ________
  2. happy → ________（注意：y 变 i 再加 -ness）
  3. sad → ________
  4. good → ________（注意：good 变 goodness）
  5. dark → ________

第3题：动词变名词（人）

把下面的动词变成表示人的名词（加 -er 或 -or）。

  1. teach → ________
  2. play → ________
  3. sing → ________
  4. dance → ________
  5. write → ________

第4题：选词填空

选择正确的词填空。

  1. She runs very ________. （quick / quickly）
  2. He is a man of ________. （kind / kindness）
  3. My mom is a ________. （teach / teacher）
  4. Please speak ________. （loud / loudly）
  5. I like the ________ of the sun. （warm / warmth）

四、参考答案

第1题：

  1. quickly
  2. slowly
  3. happily
  4. loudly
  5. carefully

第2题：

  1. kindness
  2. happiness
  3. sadness
  4. goodness
  5. darkness

第3题：

  1. teacher
  2. player
  3. singer
  4. dancer
  5. writer

第4题：

  1. quickly（修饰动词 runs，用副词）
  2. kindness（介词 of 后面用名词）
  3. teacher（表示人，用名词）
  4. loudly（修饰动词 speak，用副词）
  5. warmth（the ... of 结构，用名词）
